Detailed Description

LeptonInfo - a transient class used to consolidate useful information about a single lepton.

Constructor argument: isElectron flavor bool index index of element in d3pd vectors lv pointer to corrected lorentz vector d3pdObj either ElectronD3PDObject* or MuonD3PDObject*

Accessing variables: Use lv() to access the corrected 4-momentum
